# Employability Intelligence System

A web app that predicts student placement probability using Machine Learning.

## What it does
- Analyses 1,000 student records across CGPA, aptitude, projects, internships & soft skills
- EDA with distributions, placement rate charts & correlation heatmap
- Random Forest — 86% accuracy
- Logistic Regression — 87.5% accuracy
- Live predictor — enter your profile, get placement probability instantly
- Personalized recommendation engine

## Tech Stack
Python · Pandas · Scikit-Learn · Matplotlib · Seaborn · Streamlit

## Key Finding
Projects and Aptitude Score are stronger predictors of placement than CGPA.
